
下面不是官方难度指数，是一个“数理化体感模型”：

[  
D_A = 100 \times \prod_{i=1}^{A}(1+w_i)  
]

其中：

- **A0 = 100**
    
- (w_i) 是每一层新增惩罚带来的边际压力
    
- 指数越高，代表对牌组强度、路线规划、容错和资源管理要求越高
    

|难度|新增机制|边际压力|累计难度指数|
|---|---|--:|--:|
|A0|无进阶惩罚|—|**100**|
|A1|精英更多|+7%|**107**|
|A2|幕间治疗只回 80% 已损生命|+6%|**113**|
|A3|敌人和宝箱金币 -25%|+9%|**124**|
|A4|药水栏 -1|+7%|**132**|
|A5|开局带进阶灾祸诅咒|+12%|**148**|
|A6|休息点变少|+13%|**167**|
|A7|稀有牌、升级牌更少|+10%|**184**|
|A8|敌人更难杀|+15%|**212**|
|A9|敌人攻击更痛|+17%|**248**|
|A10|第三幕双 Boss|+20%|**297**|

### 结论

如果 **A0 = 100**，我会把 **A10 估成约 300**。  
也就是：**A10 不是比 A0 难一点，而是大约 3 倍资源压力/容错压力。**

分段看更清楚：

|区间|难度变化|主要压力|
|---|--:|---|
|A0 → A4|100 → 132|资源开始紧，但还能靠经验补|
|A5 → A7|148 → 184|牌组稳定性明显下降，诅咒、少营火、少好牌开始折磨|
|A8 → A10|212 → 297|真正高压区，敌人数值、伤害和双 Boss 要求完整成型牌组|

最大的坎通常是：

**A5、A6、A8、A9、A10。**

其中 **A10** 不是单纯“最后多打一场 Boss”，而是要求你的牌组能连续打两场高强度战斗：不能只靠一次性爆发，必须有稳定输出、稳定防御、续航或强循环。

如果你说的“第一层为 100”是指 **A1 = 100**，那换算后大概是：

**A0 = 93，A1 = 100，A5 = 138，A8 = 198，A10 = 278。**
